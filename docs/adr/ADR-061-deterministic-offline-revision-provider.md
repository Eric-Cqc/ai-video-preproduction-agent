# ADR-061 — Deterministic Offline Revision Provider

Status: Accepted for Stage 13

Stage 13 uses a deterministic fixture provider mode only. It can produce
bounded valid output and named schema, semantic, refusal, timeout and provider
failure modes for tests. No model SDK, credential, URL, network, image/video
generator or media renderer is introduced. A real provider requires a later
privacy, threat, cost and retention review.
