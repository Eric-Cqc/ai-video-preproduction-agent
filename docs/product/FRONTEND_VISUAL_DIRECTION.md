# Production Desk visual direction

The local product workspace uses a quiet editorial production-board treatment: warm paper,
deep green operational controls, restrained amber stage emphasis, and serif headings for
artifact-oriented work. It is deliberately not a chat surface or a media-generation console.

The production rail makes the immutable preproduction sequence visible: Intake, Brief,
Concepts, Script, Storyboard, Shot Plan, Review, and Delivery. The center panel is reserved for
the active project task; the right panel keeps provenance, version and next-step context close
to the work without exposing internal operation digests or persistence details.

Responsive behavior preserves the same information hierarchy. At wide widths this is a
three-column desk; at tablet width project context moves below the task; at phone width the rail
becomes a four-column stage index and forms collapse to a single column.
