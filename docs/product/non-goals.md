# 非目标

当前及基础架构阶段不做以下事情：

- 自动生成、渲染、剪辑、配音、合成、交付或发布最终成片；
- 作为数字资产管理、媒体托管、广告投放、版权清算或生产执行系统；
- 在浏览器暴露模型凭据或直连模型/媒体 Provider；
- 以聊天记录或 Prompt 文本替代结构化制作数据；
- 预先建设微服务、复杂工作流编排或多云部署。

## 冻结决定

“AI 视频前期制作”不等同于“自动成片”。未来即使接入下游制作工具，也只能通过明确导出/交接边界，而不改变本系统的真相来源。

## 可替换假设与复审触发

可在有明确产品授权、责任边界、版权/安全设计和新 ADR 后评估相邻能力。仅因模型可以生成视频或市场竞品提供渲染，不构成扩展理由。
